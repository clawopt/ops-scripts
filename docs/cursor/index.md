# Cursor教程

Cursor是AI代码编辑器，GPT-4驱动，代码生成与重构神器。

## 为什么学习Cursor？

- **GPT-4驱动**：最强大的AI模型
- **代码生成**：自然语言生成代码
- **智能重构**：自动优化代码结构
- **IDE集成**：VS Code风格界面

## 核心功能

- **AI代码生成**：描述需求生成代码
- **智能补全**：上下文感知的代码建议
- **代码重构**：一键优化代码
- **多文件编辑**：跨文件代码修改
