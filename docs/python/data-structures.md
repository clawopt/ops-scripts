# 数据结构

本章介绍 Python 的内置数据结构，包括列表、字典、集合和元组。

## 学习目标

- 掌握列表的创建、访问和操作
- 理解字典的键值对结构
- 学会使用集合进行去重和集合运算
- 了解元组的不可变特性

## 数据结构概览

| 数据结构 | 特点 | 常用场景 |
|---------|------|---------|
| 列表 | 有序、可变 | 存储序列数据 |
| 字典 | 键值对、可变 | 配置管理、数据映射 |
| 集合 | 无序、唯一 | 去重、集合运算 |
| 元组 | 有序、不可变 | 函数返回多值 |
