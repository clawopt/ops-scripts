# Claude Code 简介

## 什么是 Claude Code？

Claude Code 是一个直接在你的电脑上工作的 AI 助手：

- **不仅仅是编程** - 它是由 Claude (Opus 4.5) 支持的通用助手
- **读取文件、分析数据、创建文档、运行命令**
- **运行在终端中** - 但你只需要进行对话
- **你唯一需要知道的命令**：`claude`

## 本课程适合谁？

- 想要有效使用 AI 的非技术人员
- **无需编程经验**
- **无需终端经验**
- 只需要好奇心和尝试新事物的意愿

## 课程结构

### 模块 1：基础（约 3 小时）

- **文件操作**：使用 `@` 符号读取、转换和创建文档
- **可视化工作区**：设置分屏工作流，在 Claude 工作时查看文件
- **并行代理**：启动多个 Claude 实例同时处理文件
- **自定义子代理**：创建具有不同视角的永久 AI 顾问
- **项目记忆**：使用 `CLAUDE.md` 让 Claude 记住你的项目上下文
- **命令和快捷键**：斜杠命令、键盘快捷键、高级用户功能

### 模块 2：Vibe Coding（约 1-2 小时）

- **设置和心态**：Vibe Coding 方法 - 你描述，Claude 构建
- **规划**：在构建之前通过访谈过程定义清晰的需求
- **构建和迭代**：搭建应用框架并通过截图进行优化
- **版本控制**：使用 GitHub 备份你的工作
- **上线**：部署到 Vercel，获得任何人都可以访问的真实 URL

## 要求

- Claude Pro 或 Max 订阅（claude.ai 上 $20/月）
- Mac、Windows 或 Linux 电脑
- 模块 1 大约需要 3 小时

## Claude Code vs ChatGPT

**区别**：Claude Code 不仅仅是提供建议 - 它会和你一起完成工作。

## 心态转变

从"我需要学习命令"到"我需要这个结果"

- 你正在学习如何协调，而不是编程
- 描述你想要什么，Claude 会弄清楚怎么做
- 终端只是你进行对话的地方
